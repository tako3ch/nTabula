## タスク概要

`Sources/Views/TabBarView.swift` に SwiftUI `.draggable` / `.dropDestination` (macOS 14+ ネイティブ API) を追加し、横タブバーでのタブ D&D 並び替えを実装する。ドロップ時に `AppState.moveTab()` を呼び出す。ピン留め境界の制約は AppState.moveTab() 内で処理済み。


### 2. .dropDestination 修飾子の追加 🔵

**信頼性**: 🔵 *architecture.md D&D設計より*

```swift
.dropDestination(for: UUID.self) { droppedIDs, location in
    guard let draggedID = droppedIDs.first else { return false }

    // draggedID のタブを、このタブの直後に移動
    if let fromIndex = appState.sortedTabs.firstIndex(where: { $0.id == draggedID }),
       let toIndex = appState.sortedTabs.firstIndex(where: { $0.id == tab.id }) {
        let destination = toIndex < fromIndex ? toIndex : toIndex + 1
        appState.moveTab(fromOffsets: IndexSet([fromIndex]), toOffset: destination)
    }
    return true
}
```

## 注意事項

- `macOS(14.0, *)` の API なので `@available(macOS 14.0, *)` チェックは不要（Deployment Target が 14.0）
