## 実装詳細

### 1. List の .onMove を使用した D&D 🔵

**信頼性**: 🔵 *architecture.md VerticalSidebarView D&D設計より*

**実装ファイル**: `Sources/Views/VerticalSidebarView.swift`

VerticalSidebarView が `List` を使用している場合、`.onMove` が最もシンプル:

```swift
List {
    ForEach(appState.sortedTabs) { tab in
        SidebarTabRow(tab: tab)
    }
    .onMove { fromOffsets, toOffset in
        appState.moveTab(fromOffsets: fromOffsets, toOffset: toOffset)
    }
}
```

### 2. .draggable / .dropDestination を使用した D&D 🔵

**信頼性**: 🔵 *TASK-0007 と統一した実装アプローチより*

`List` を使わない独自レイアウトの場合は TASK-0007 と同様の方法:

```swift
ForEach(appState.sortedTabs) { tab in
    SidebarTabRow(tab: tab)
        .draggable(tab.id)
        .dropDestination(for: UUID.self) { droppedIDs, location in
            guard let draggedID = droppedIDs.first else { return false }
            // TASK-0007 と同じロジック
            // ...
            return true
        }
}
```

**注意**: VerticalSidebarView の現在の実装スタイルを確認して適切な方を選択する。
