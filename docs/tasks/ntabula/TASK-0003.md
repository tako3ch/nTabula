## 実装詳細

### 1. moveTab メソッド追加 🔵

**信頼性**: 🔵 *interfaces.swift AppState.moveTab()より*

**実装ファイル**: `Sources/App/AppState.swift`

```swift
func moveTab(fromOffsets: IndexSet, toOffset: Int) {
    // 移動元・移動先のピン留め状態を確認
    guard let fromIndex = fromOffsets.first else { return }

    let movingTab = tabs[fromIndex]
    let destinationIsPinned: Bool

    if toOffset < tabs.count {
        destinationIsPinned = tabs[toOffset].isPinned
    } else {
        // 末尾への移動: 末尾タブのピン状態に合わせる
        destinationIsPinned = tabs.last?.isPinned ?? false
    }

    // ピン留め境界をまたぐ移動は禁止
    guard movingTab.isPinned == destinationIsPinned else { return }

    tabs.move(fromOffsets: fromOffsets, toOffset: toOffset)
    saveTabs()
}
```

### 2. sortedTabs との整合性 🔵

**信頼性**: 🔵 *AppState.sortedTabs（ピン先頭ソート）より*

- `moveTab` は `tabs` 配列を直接操作
- `sortedTabs` は `tabs` から computed で生成（ピン先頭ソート）
- D&D UI では `sortedTabs` を表示し、ドロップ時に元の `tabs` インデックスを計算

## 注意事項

- `sortedTabs` はピン先頭ソート済みの computed プロパティのため、D&D UI では sortedTabs の表示インデックスと tabs 配列インデックスのマッピングが必要
