## 実装詳細

### 1. CommandGroup への追加 🔵

**信頼性**: 🔵 *architecture.md Cmd+W設計より*

**実装ファイル**: `Sources/App/nTabulaApp.swift`

```swift
CommandGroup(replacing: .newItem) {
    Button("新規タブ") {
        appState.addNewTab()
    }
    .keyboardShortcut("t", modifiers: .command)

    // 追加
    Button("タブを閉じる") {
        if let tab = appState.activeTab {
            appState.closeTab(tab)
        }
    }
    .keyboardShortcut("w", modifiers: .command)
    .disabled(appState.activeTab == nil)
}
```

### 2. ピン留めタブの保護 🔵

**信頼性**: 🔵 *REQ-108・dataflow.md フロー9より*

ピン留めタブの保護は `AppState.closeTab(_:)` 内の既存実装が担う:

```swift
func closeTab(_ tab: TabItem) {
    guard !tab.isPinned else { return }  // 既存ガード
    // ...
}
```

追加実装不要。
