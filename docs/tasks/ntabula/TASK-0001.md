## タスク概要

`Sources/Utilities/PersistenceManager.swift` に macOS Keychain アクセスメソッドを追加する。Security.framework の SecItem API（SecItemAdd / SecItemCopyMatching / SecItemDelete）を直接実装し、Notion Integration Token を Keychain に安全に保存・読み込みできるようにする。

既存の UserDefaults 平文保存から Keychain 保存へのマイグレーション基盤を提供する。起動時マイグレーションロジックは TASK-0002 で実装する。

### 3. saveToken メソッド実装 🔵

**信頼性**: 🔵 *persistence-schema.md Keychain操作メソッドより*

```swift
func saveToken(_ token: String) {
    guard let data = token.data(using: .utf8) else { return }

    let query: [String: Any] = [
        kSecClass as String: kSecClassGenericPassword,
        kSecAttrService as String: keychainService,
        kSecAttrAccount as String: keychainAccount
    ]
    // 既存エントリを削除してから追加（上書き）
    SecItemDelete(query as CFDictionary)

    var addQuery = query
    addQuery[kSecValueData as String] = data
    SecItemAdd(addQuery as CFDictionary, nil)
}
```

### 4. loadToken メソッド実装 🔵

**信頼性**: 🔵 *persistence-schema.md 起動時マイグレーションより*

起動時マイグレーションロジック（UserDefaults → Keychain）を含む:

```swift
func loadToken() -> String {
    let query: [String: Any] = [
        kSecClass as String: kSecClassGenericPassword,
        kSecAttrService as String: keychainService,
        kSecAttrAccount as String: keychainAccount,
        kSecReturnData as String: true,
        kSecMatchLimit as String: kSecMatchLimitOne
    ]

    var result: AnyObject?
    let status = SecItemCopyMatching(query as CFDictionary, &result)

    if status == errSecSuccess,
       let data = result as? Data,
       let token = String(data: data, encoding: .utf8) {
        return token
    }

    // Keychain にない場合: UserDefaults マイグレーション
    let legacyToken = defaults.string(forKey: "nTabula.notionToken") ?? ""
    if !legacyToken.isEmpty {
        saveToken(legacyToken)
        defaults.removeObject(forKey: "nTabula.notionToken")
        return legacyToken
    }

    return ""
}
```
